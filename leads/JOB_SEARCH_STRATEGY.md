## üéØ Resume Tailoring Tips

### For Finance Roles:
- Highlight: Excel, financial modeling, analysis
- Keywords: Bloomberg, valuation, financial statements
- Projects: Investment analysis, portfolio tracking

### For CS Roles:
- Highlight: Python, JavaScript, Git, data structures
- Keywords: APIs, databases, algorithms, testing
- Projects: GitHub repos, hackathons, coursework

### For Both:
- Quantify everything ("improved X by Y%")
- One page MAX
- PDF format
- Clean, professional design
